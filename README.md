Défi :
Créer une galerie responsive d'images parfaitement centrée en utilisant uniquement Flexbox.

Débutant Niveau 1
1. Crée un container principal <div> avec la classe .gallery
2. À l'intérieur, ajoute 6 images (tu peux utiliser des placeholders comme https://picsum.photos/200/300?random=1)
3. Centre horizontalement toutes les images dans le container
4. Donne un espace de 20px entre les images

Intermédiaire Niveau 2
1. Fais en sorte que les images se disposent sur plusieurs lignes si nécessaire
2. Centre les lignes entre elles verticalement
3. Quand la page devient petite, réduis l'espace entre les images à 10px
4. Centre le titre de la page horizontalement et verticalement dans le header

Avancé Niveau 3
1. Crée un layout responsive où :
2. Sur grand écran (> 1200px) : 4 images par ligne
3. Sur tablette (768px-1200px) : 3 images par ligne
4. Sur mobile (< 768px) : 2 images par ligne
5. Ajoute un effet au survol des images (zoom léger + ombre)
6. Centre tout le contenu de la page (header, gallery, footer)
7. Fais en sorte que le footer reste toujours en bas de page même si le contenu est court
